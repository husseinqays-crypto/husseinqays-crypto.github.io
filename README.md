# exams-site

موقع بسيط باللغة العربية لعرض **تواريخ الامتحانات** مع شرح لكل امتحان.
ارفع هذا المجلد إلى مستودع GitHub باسم: `اسم_مستخدمك.github.io` ثم فعّل GitHub Pages.

## تعديل الامتحانات
- افتح الملف: `data/exams.json`
- كل عنصر داخل المصفوفة يمثل امتحانًا (يمكنك النسخ واللصق وتغيير القيم).
- الحقول:
  - `title` اسم المادة
  - `date` التاريخ بصيغة `YYYY-MM-DD`
  - `time` الوقت بصيغة `HH:MM` (اختياري)
  - `location` القاعة
  - `grade` المرحلة/السنة
  - `duration_minutes` المدة بالدقائق
  - `description` وصف قصير
  - `notes` ملاحظات (اختياري)
  - `more_info_url` رابط تفاصيل إضافية (اختياري)

## النشر على GitHub Pages
1. ادخل إعدادات المستودع Settings → Pages
2. اجعل النشر من الفرع `main` والمجلد `/ (root)`
3. بعد لحظات سيعمل موقعك على: `https://اسم_مستخدمك.github.io`
